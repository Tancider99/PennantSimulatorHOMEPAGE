# ⚾ Pennant Simulator 2027（ペナントシミュレーター2027）

本格的なNPB（日本プロ野球）チーム設計・ペナントレースシミュレーションゲームです。
洗練されたプロフェッショナルなUIと安定したゲームプレイを実現。

## ✨ 特徴

- 🏟️ **12球団対応** - セ・パ両リーグの全12球団から選択可能
- 📊 **詳細な選手能力** - 打撃、守備、投球能力を細かくシミュレート
- 📅 **リアルなシーズン** - 143試合のペナントレースを戦い抜く
- 🎯 **戦略的采配** - 打順設定、先発投手選択、選手補強
- 🌟 **ドラフト会議** - シーズン終了後に新戦力を獲得
- 🌍 **外国人補強** - FA市場から即戦力を獲得

## 🖥️ プロフェッショナルUI

- ダークモードベースのモダンデザイン
- スムーズなアニメーションとトランジション
- チームカラーに対応したカスタマイズ表示
- 直感的な操作性

## 📋 必要環境

- Python 3.8以上
- pygame 2.0以上

## 🚀 インストール

```bash
pip install -r requirements.txt
```

## ▶️ 起動方法

```bash
python main.py
```

## 🎯 ゲームの流れ

1. **タイトル画面** - 「ゲームスタート」をクリック
2. **難易度選択** - イージー〜エキスパートから選択
3. **チーム選択** - 監督するチームを選ぶ
4. **メインメニュー** - 各種機能にアクセス
   - 📋 オーダー設定 - 打順と先発投手を決定
   - 📅 試合日程 - シーズンのスケジュールを確認
   - ▶️ 試合開始 - 次の試合をシミュレート
   - 🏆 順位表 - 現在の順位を確認
   - 🌍 外国人補強 - FA市場から選手を獲得

## 🎨 UIデザインシステム

### カラーパレット
- **背景**: ダークグレー系 (#0C0E12, #161A20)
- **アクセント**: ブルー (#3B82F6), グリーン (#22C55E), レッド (#EF4444)
- **テキスト**: ホワイト系 (#F8FAFC, #94A3B8)

### コンポーネント
- `Button` - アニメーション付きボタン（primary/success/danger/ghost/outline）
- `Card` - 情報表示用カード
- `ProgressBar` - プログレスバー
- `Table` - データテーブル
- `Toast` - 通知メッセージ
- `RadarChart` - 能力レーダーチャート

## 🏆 高度な機能

### 特殊能力システム
パワプロ風の特殊能力を実装
- **打撃系**: アベレージヒッター、パワーヒッター、チャンスなど
- **投球系**: 奪三振、制球力、重い球など
- **守備系**: 強肩、守備職人など
- **走塁系**: 盗塁、走塁など

### 選手育成システム
- 練習メニューによる能力向上
- 成長曲線（若手は成長しやすい）
- 疲労管理

### 詳細試合エンジン
- 打席ごとのシミュレーション
- 投手-打者の相性計算
- 状況に応じた采配
